## Structure

- **Source/** - Original model files (STL, STEP formats)
- **Edited/** - Modified and recreated parts
  - **Modified/** - Parts modified and other improvements
  - **Recreated/** - Experimental recreations (see warning below)
- **3DPrint/** - Ready-to-print 3MF files
